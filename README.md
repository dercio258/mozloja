# Lite Payment Project

A simplified dashboard and payment application.

## Setup

1.  Navigate to the directory:
    ```bash
    cd lite_project
    ```

2.  Install dependencies:
    ```bash
    npm install
    ```
    *(Dependencies: express, ejs, dotenv, body-parser, axios)*

3.  Configure `.env` file with your credentials and API keys.

## Running

Start the server:
```bash
node server.js
```

Visit `http://localhost:3000` in your browser.
